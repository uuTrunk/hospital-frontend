# 医院管理系统

## 项目描述
这是一个基于Vue2开发的医院管理系统，主要实现了患者入院评估功能。

## 功能特点
1. 患者入院评估管理：包括健康评估和首次照护评估
2. 评估状态管理：
   - 未完成评估显示待评估图标
   - 已完成评估显示完成图标和评估时间
   - 不同意入院的评估显示叹号提示
3. 数据条件筛选：可通过姓名、入院编号和日期范围筛选数据
4. 评估数据编辑权限控制：未完成最终入院前，可修改评估数据
5. 智能排序：待评估的条目置顶，已完成评估的条目按登记时间降序排列
6. 页签功能：显示待处理人数

## 技术栈
- 前端框架：Vue.js 2.x
- 状态管理：Vuex
- 路由管理：Vue Router
- UI设计：自定义CSS样式

## 安装与运行
```bash
# 安装依赖
npm install

# 本地开发环境运行
npm run serve

# 构建生产环境版本
npm run build
```
